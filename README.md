## 🛠️ Troubleshooting

### The page does not open

- Check that you opened the main file in the project folder.

### Images do not appear

- Confirm the image files are in the right folder.
- Check that file names match the references in the project.
- Refresh the browser page.

### The layout looks broken

- Use a modern browser.
- Make sure all CSS files are present.
- Do not move files out of the original folder structure.
